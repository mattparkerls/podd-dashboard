ตารางสรุปการรายงานในแต่ละอปท.

  • รูปแบบของข้อมูลในตารางระบุค่าของจำนวนของรายงานไม่พบเหตุผิดปกติ, จำนวนของรายงานพบเหตุผิดปกติ ตัวอย่างของข้อมูล:

    • 0 , 0 คือ จำนวนของรายงานไม่พบเหตุผิดปกติเท่ากับ 0 และ จำนวนของรายงานพบเหตุผิดปกติเท่ากับ 0

      โดยที่วงกลมสีเทาจะแสดงบนจำนวนรายงานไม่พบเหตุผิดปกติเมื่อจำนวนรายงานเท่ากับ 0 และวงกลมสีเทาจะแสดงบนจำนวนรายงานพบเหตุผิดปกติเมื่อจำนวนรายงานเท่ากับ 0

    • 1 , 2 คือ จำนวนของรายงานไม่พบเหตุผิดปกติเท่ากับ 1 และ จำนวนของรายงานพบเหตุผิดปกติเท่ากับ 2

      โดยที่วงกลมสีเขียวจะแสดงบนจำนวนรายงานไม่พบเหตุผิดปกติเมื่อจำนวนรายงานมากกว่า 0 และวงกลมสีแดงจะแสดงบนจำนวนรายงานพบเหตุผิดปกติเมื่อจำนวนรายงานมากกว่า 0
